# Roadmap

## v0.6.0 — управляемый цикл
- Явный оркестратор цикла (state machine + таймауты).
- Чёткие переходы: enter → monitor → close → cooldown.
- Отдельные режимы тест/лайв с защитами.

## v0.7.x — алгоритм
- Автоматическая логика входа/выхода.
- Настраиваемые параметры риска и объёма.
- Улучшенный PnL (комиссии, частичные исполнения).

## v1.0 — хедж-профили
- Набор готовых профилей хеджа.
- Сохранение/загрузка профилей в GUI.
- Расширенная телеметрия и отчёты.
