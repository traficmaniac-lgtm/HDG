# Roadmap

## v0.6.0 — инструменты для алгоритма
- Формализация state machine и таймаутов цикла.
- Явные контракты данных и статусов (market/order/position).
- Защитные проверки и телеметрия для безопасного запуска цикла.

## v0.7.x — первый детерминированный цикл
- Детерминированная последовательность enter → monitor → close.
- Минимальная параметризация для повторяемого поведения.

## Что НЕ делаем до v0.6.0
- Автоматические решения входа/выхода.
- Авто-повторяющиеся циклы.
- Расчёты PnL с комиссиями/просадками/процентами.
