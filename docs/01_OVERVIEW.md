# Overview

## Назначение проекта
Directional Hedge Scalper — GUI-клиент для ручного тестового цикла BUY → SELL на Binance Cross Margin.

## Что это НЕ
- НЕ автотрейдинг.
- НЕ стратегия/алгоритм принятия решений.
- НЕ цикл, который сам стартует/повторяется.

## Текущий статус
- Версия: **v0.5.8**.

## Что уже работает (факты)
- GUI на PySide6, запуск через `python -m src.app`.
- Получение котировок через WS bookTicker с HTTP fallback.
- Роутер котировок с метриками свежести и источника.
- REST-доступ к Binance: server time, exchangeInfo, margin account, open orders, создание/отмена ордеров.
- Ручной запуск BUY и ручное закрытие SELL из GUI.
- Трекинг активных ордеров, статусов, позиции и простого PnL в интерфейсе.
- Настройки сохраняются в `config/settings.json`.

## Что запрещено делать алгоритму на этом этапе
- Самостоятельно принимать решения о входе/выходе.
- Запускать автоматические циклы/повторы.
- Параллельно размещать несколько торговых действий.
- Использовать логику, не подтверждённую текущей реализацией GUI/TradeExecutor.
