# Overview

## Назначение проекта
Directional Hedge Scalper — GUI-клиент для ручного тестового цикла BUY → SELL на Binance.

## Что делает бот
- Подключает Market Data и показывает котировки/статусы.
- Позволяет вручную инициировать BUY и вручную закрыть SELL.
- Отображает состояние ордеров, позиции и простой PnL.

## Чего НЕ делает бот
- Не принимает торговые решения и не запускает стратегию.
- Не запускает автоматический цикл сам по себе.
- Не выполняет параллельные торговые действия.

## Торговый инструмент и режим
- Инструмент: **EURIUSDT**.
- Режим: **Cross Margin x3**.
