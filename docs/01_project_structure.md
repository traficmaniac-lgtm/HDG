# Структура проекта

Этот документ описывает основные директории и то, как они связаны между собой.

## Верхний уровень

- `README.md` — краткая инструкция по запуску и назначению проекта.
- `requirements.txt` — зависимости Python.
- `RUN_GUI.ps1` — сценарий запуска GUI в Windows.
- `.env.example` — пример переменных окружения.
- `src/` — исходный код приложения.
- `docs/` — документация по архитектуре и алгоритмам.

## Директория `src/`

- `app/` — точка входа GUI и запуск приложения.
- `core/` — модели данных, конфигурации, логирование, конечный автомат и торговый движок.
- `engine/` — реализация торгового цикла и сценариев работы алгоритма.
- `exchange/` — слой исполнения ордеров и взаимодействия с биржей.
- `services/` — клиенты REST/WS и вспомогательные источники данных.
- `gui/` — пользовательский интерфейс и виджеты.

## Поток данных высокого уровня

1. GUI инициирует подключение и параметры стратегии.
2. `core.trade_engine.TradeEngine` держит состояние и координирует исполнение.
3. `engine.directional_cycle.DirectionalCycle` реализует торговый цикл.
4. `exchange.binance_margin.BinanceMarginExecution` исполняет заявки через REST.
5. `services` получает рыночные данные (WS/HTTP) и даёт их в `TradeEngine`.
