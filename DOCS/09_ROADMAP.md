# Roadmap

## Текущее состояние (по коду)
- Детализированный цикл BUY → SELL с конечным автоматом `TradeExecutor`.
- Follow‑top репрайс входа с фильтрами стабильности источника.
- Авто‑выход по TP/SL в тиках и ручной CLOSE.
- Таймауты входа/выхода, TTL SELL и повторные попытки.
- OrderTracker для частичных/полных исполнений.

## Ближайшие потенциальные улучшения
- Явное управление стратегией входа (сигналы), отделённое от текущего цикла.
- Расчёт PnL с учётом комиссий/процентов/проскальзывания.
- Расширенная работа с частичными исполнениями (усреднение/ступенчатый выход).
- Отчёты по сессиям и экспорт истории сделок.
