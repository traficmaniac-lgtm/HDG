# Directional Hedge Scalper — обзор

## Название проекта
Directional Hedge Scalper (DHS)

## Цель проекта
Создать стабильный торговый бот для **быстрого направленного скальпинга** в режиме Cross Margin, который:
- открывает позицию по заранее определённой логике,
- закрывает её по TP/SL или вручную,
- обеспечивает предсказуемость исполнения и учёт PnL,
- минимизирует влияние проблем с источниками данных.

## Как извлекается прибыль (кратко)
- Движение **в одном направлении**: вход по BUY (скальперный лимит/маркет), выход SELL.
- Прибыль фиксируется **на малых ценовых шагах** (ticks) через Take Profit.
- Риск ограничивается Stop Loss и аварийными блокировками (data_blind, ошибки ордеров).

## Почему выбран EURIUSDT
- Низкая волатильность и «узкий» ценовой диапазон позволяют стабильный скальпинг.
- Небольший tick size даёт тонкую настройку TP/SL.
- Символ достаточно ликвиден для быстрых циклов и проверки логики бота.

## Текущий статус
- **Версия:** v0.7.x (PRE v0.8.0)
- **Состояние:** основная механика (GUI, источники цены, торговый движок, расчёт PnL) работает, но требуется формализация полной FSM и улучшение устойчивости жизненного цикла ордеров для перехода к v0.8.0.
